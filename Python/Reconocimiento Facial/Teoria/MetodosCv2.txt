cvtColor(src, code) => Agrega una gama de colores a una imagen, se puede 
usar para transformar a  escala de grises.

threshold(src, tresh, maxval, type) => Se utiliza para aplicar el umbral. 
Retorna dos valores, el umbral utilizado y la imagen con el umbral aplicado.

findContours(umbral, mode, method) => Encuentra el contorno de una imagen.
